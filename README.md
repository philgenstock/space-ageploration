# Space Ageploration

A Factorio mod that enhances Space Age progression with new technologies and buildings across Fulgora, Vulcanus, Gleba, and Aquilo.

## Requirements

- Factorio 2.0 or later
- Space Age DLC

## Planned Features

### Fulgora
- **Tier 1**: Holmium Productivity, Holmium Circuits
- **Tier 2**: Pylon
- **Tier 3**: Robo Pylon, Supercharger

### Vulcanus
- **Tier 1**: Ducts, Large Steam Turbine
- **Tier 2**: Manufactory, Large Beacon
- **Tier 3**: Industrial Rocket Silo

### Gleba
- **Tier 1**: Liquid Nutrients
- **Tier 2**: Universal Soil
- **Tier 3**: Smart Rockets

### Aquilo
- **Tier 1**: Freeze Resistant Pipes
- **Tier 2**: Deep-Freezing I (Bioflux & Science), Deep-Freezing II (Eggs)
- **Tier 3**: Cooled Computing

### Combo Technologies
- **Fulgora & Aquilo**: Fusion Robots
- **Fulgora & Vulcanus & Gleba**: Research Station

### Special Buildings
- Computer (requires Space Science)
- Super Computer (requires All Tier 1 techs)
- Quantum Computer (requires All Tier 3 techs)


## Development Status

This mod is currently in early development. The basic structure is in place, but most features are not yet implemented.
